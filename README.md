    결과 : 실패

    애초에 비율을 3:1로 조정해놓은걸 임의로 샘플링해서 생긴 문제이므로

    두가지 해결책 마련

    2) 테스트 데이터도 샘플링 : 3만 : 1만으로 3대 1비율 유지 및 가동 가능성 있음. 정확도 떨어질 것으로 예상.
